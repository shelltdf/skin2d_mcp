# 详细设计：编辑器壳

## 布局结构

- 根容器：`display: flex; flex-direction: column; height: 100vh`，字体 `Segoe UI Variable` / `Segoe UI`。
- **标题行**：应用名「Skin2D」、窗口控制占位（最小化/最大化/关闭仅视觉，浏览器内不连接系统 API）。
- **菜单栏**：文件（新建、打开、**导入…**、退出）、编辑、视图、帮助。
- **工具条**：图标按钮（导入、Dock 显示/隐藏、画布全屏、显示器全屏），导入按钮悬浮提示（点击后消失）。
- **主体 Dock**：左（层级）/中（视口）/右（属性）/底（时间轴）四区；支持拖拽分隔条调整宽高与显示/隐藏；底部时间轴 Dock 横跨窗口全宽。
- **底部状态栏**：显示最近日志摘要，点击打开日志窗口。

## 主题

- 使用 CSS 变量：`--bg`、`--panel`、`--border`、`--accent`、`--text`。
- 浅色方案默认，与 Windows 11 浅色对齐。
- 支持主题设置：跟随系统 / 浅色 / 深色；通过 `data-theme` 应用到 DOM，并持久化到 localStorage。
- 支持语言设置：中文 / 英文；所有固定 UI 文案通过 `t(zh,en)` 切换（对象名/专有名词保持原样）。

## 交互

- 「导入」打开 `<input type="file">`，多选关闭；将 `File` 交给导入管线。
- 「画布全屏」：隐藏侧栏与时间轴 Dock；Esc 可退出。
- 「显示器全屏」：仅请求中心画布容器进入浏览器全屏；退出时恢复进入前的 Dock 可见性快照。
- 状态栏点击：打开日志窗口（纯文本控制台）。
