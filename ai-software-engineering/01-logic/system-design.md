# 系统设计

## 子系统

1. **编辑器壳**：菜单、工具条、可折叠面板、状态栏；Fluent 风格令牌（颜色、圆角、间距）。
2. **文档模型**：工程、骨骼树、插槽、动画列表（MVP 可为导入摘要 + 占位）。
3. **导入管线**：文件扩展名与 JSON 结构探测 → 对应解析器 → 统一 `ImportResult`。
4. **视口**：Spine canvas 绘制与 Live2D WebGL 视口（互斥显示）；显示层开关；画布选中与高亮。
5. **时间轴**：播放/暂停、寻帧、Dope Sheet/Curves 只读预览。
6. **设置与日志**：语言/主题（持久化 + DOM 应用），应用日志（状态栏 + 日志窗口）。

## 依赖关系

```
编辑器壳 → Pinia stores → 视口 / 层级面板
                ↓
           导入管线（独立模块，可单测）
```
