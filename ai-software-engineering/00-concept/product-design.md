# 产品设计：Skin2D — Windows 风格 2D 骨骼蒙皮动画编辑器

## 愿景

为独立开发者与小型团队提供一款在浏览器中运行的 2D 骨骼蒙皮动画编辑工具，界面遵循 **Windows 11 / Fluent** 视觉与交互习惯（标题栏、命令区、可停靠面板、清晰层次），并支持从**主流开放或可互操作格式**导入资产，便于与现有流水线衔接。

## 目标用户

- 需要编辑 2D 骨骼层级、权重与动画时间轴的游戏/工具作者。
- 希望在 Windows 桌面浏览器或打包为桌面壳（如 Electron/Tauri，后续可选）中使用的用户。

## 核心用例

1. **新建/打开工程**：管理纹理、骨骼与动画剪辑。
2. **编辑骨骼与蒙皮**：在画布中操作骨骼变换，调整顶点权重（后续迭代）。
3. **编辑时间轴**：关键帧、曲线或步进（依实现阶段）。
4. **导入**：从开放格式读取骨架、网格/区域与动画数据，映射到内部模型。
5. **预览与导出**（后续）：预览播放；导出为内部或开放格式。

## 界面概述（非几何权威）

- **顶区**：应用标题、主菜单（文件/编辑/视图/帮助）、可选工具条。
- **左**：层级树（骨骼/插槽/附件）。
- **中**：主视口（2D 画布，网格与骨骼叠加）。
- **右**：属性与变换。
- **底**：时间轴与播放控制。

详细控件布局与像素几何以实现侧 `App.vue` 布局与主题变量为准；若需固定 UI 示意图，可后续补充 `ui.svg`。

## 成功标准（MVP）

- 可运行的单页应用，呈现 Windows 风格外壳与空白/占位视口。
- 提供「导入」入口，并对至少一种开放格式完成 **解析 → 内部模型占位展示**（文件名与根对象信息）。
- 文档与 `02-physical/skin2d-editor-web/spec.md` 中对导入格式与行为一致。
