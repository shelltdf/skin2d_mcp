# 用户说明书：Skin2D 编辑器（Web）

## 环境前提

- 现代浏览器（Chrome、Edge、Firefox 最新稳定版）。
- Windows 10/11 推荐；其他操作系统亦可运行 Web 版。

## 启动

1. 开发者构建并托管 `skin2d-editor/dist/`（见开发维护说明书），或运行开发服务器后访问本地 URL。
2. 打开页面后即进入编辑器主界面。

## 基本操作

- **文件 → 导入…**：可选单文件或多选。
  - **Spine 贴图 + 动画**：一次多选同一角色的 `*.json`（骨架）+ `*.atlas` + 图集中引用的贴图（如 `*.png`）；导入后画布显示角色，底部时间轴可选动画并播放/暂停。
  - **Live2D Cubism（画布预览）**：将模型目录打成 **单个 `*.zip`**（含 `*.model3.json` / `.moc3` / 贴图等），导入时只选这一份 zip，即可在画布区预览。
  - **Live2D Cubism（仅元数据）**：导入 `*.model3.json`（UTF-8 JSON）可查看模型元数据与动作组名，但不会渲染 Live2D 画布；不要单独只选 `.moc3`。
  - 其他：`.json`（Spine / DragonBones）、`.dbproj`（DragonBones 工程）、`.gltf` / `.glb`（摘要）。
- **帮助 → 支持的文件格式…**：查看各格式的说明、版本与兼容性注意（以对话框内文案为准）。
- 导入成功后，右侧属性区与中央视口会显示骨架摘要信息；Spine 多文件包另显示运行时绘制。
- **时间轴（Timeline）**：可播放/暂停、拖动进度条寻帧；支持 Dope Sheet 与曲线预览（只读）。
- **画布全屏 / 显示器全屏**：工具条右侧按钮可进入画布全屏（隐藏 Dock）与显示器全屏（仅全屏画布区域）。
- **状态栏与日志**：窗口底部状态栏显示最近状态；点击可打开日志窗口（纯文本），用于排查导入/预览/全屏等问题。

## 常见问题

- **无法识别文件**：确认是否为支持的 JSON 结构或 glTF；查看页面内警告信息。
- **Spine 只有线框无贴图**：请用多选导入 JSON + atlas + 贴图；仅单 JSON 不会加载图集。
