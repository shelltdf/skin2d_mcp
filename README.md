# skin2d_mcp

基于 Vue 3 的 **2D 骨骼蒙皮动画编辑器**（Windows 11 风格界面），支持主流开放/互操作格式导入（Spine JSON、DragonBones、glTF 2.0）。

## 仓库结构

- `ai-software-engineering/`：四阶段工程文档（概念 / 逻辑 / 物理 / 运维）。
- `skin2d-editor/`：前端实现（Vite + Vue 3 + TypeScript + Pinia）。

## 快速开始

```bash
cd skin2d-editor
npm install
npm run dev
```

或使用仓库根目录 Python 封装：`python dev.py`（见 `build.py` / `test.py` / `publish.py`）。
